# Tulz.net

A collection of free, fast, and privacy-focused online developer tools.

## Tools Included
1.  **Password Generator**: Create secure, random passwords.
2.  **Word Counter**: Count words, characters, sentences, and paragraphs.
3.  **Age Calculator**: Calculate exact age from birthdate.
4.  **BMI Calculator**: Body Mass Index calculator with health categories.
5.  **Color Converter**: RGB to Hex and Hex to RGB.
6.  **Lorem Ipsum Generator**: Placeholder text generator.
7.  **Timestamp Converter**: Unix Epoch to Human date converter.

## Technologies
- **HTML5**
- **CSS3** (Responsive, Dark Mode)
- **Vanilla JavaScript** (No libraries/frameworks)

## Setup
Simply open `index.html` in your browser.
